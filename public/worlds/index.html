Demos

Currently we've developed 3 demo worlds.
1) The Montane Forest World allows students to explore and learn about the alpine biosphere in a lush 3d world.
2) The Robotics and Automation world allows students to interact with a simulated robot to better understand control signals and reference frames
3) The Class world simulates a virtual classroom where students can attend virtual classes in a 3D environment.
